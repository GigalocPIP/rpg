# A Lenda de Sir Cuzcuz

Jogo retrô com herói animado, inimigos com IA, música de fundo, efeitos sonoros e sistema de vida.

## Como rodar

1. Instale as dependências:
```
npm install
```

2. Inicie o projeto:
```
npm start
```

3. Suba o conteúdo para o GitHub e use no Netlify para publicação online.
